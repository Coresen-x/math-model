# 时间序列平滑模型

## Simple模型（简单指数平滑法）

* **适用条件**：不含趋势和季节成分
* **类似的ARIMA模型**：ARIMA（0，1，1）
* **说明**：

  * Spss的专家建模如果选择了Simple模型用来估计，那么软件会帮我们自动选取一个适合的平滑系数α使得预测误差最小
  * 只能预测一期，由公式决定

## 阻尼趋势模型

* **适用条件**：线性趋势逐渐减弱且不含季节成分
* **类似的ARIMA模型**：ARIMA（1，1，2）

## 简单季节性

* **适用条件**：含有稳定的季节成分、不含趋势

## 温特加法模型

* **适用条件**：含有线性趋势和稳定的季节成分

## 温特乘法模型

* **适用条件**：含有线性趋势和不稳定的季节成分
